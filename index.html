malloc by wedesoft

Ruby-extension for raw memory allocation and pointer operations

This gem defines the class Malloc. Malloc#new allows you to allocate memory, using Malloc#+ one can do pointer manipulation, and Malloc#read and Malloc#write provide reading Ruby strings from memory and writing Ruby strings to memory.

Download

If you have set up gemcutter, you can install malloc simply by running: 

$ gem install malloc

Note that malloc so far was only tested under GNU/Linux (ruby 1.8.6 and ruby 1.9.1p243). malloc furthermore requires GNU C++ to build. Let me know if it runs on other platforms already.

Documentation

At the moment there is no documentation. Please refer to the documentation of HornetsEye::Malloc instead.
